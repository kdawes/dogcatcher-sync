Trivial example of a full stack - mobile / api / backend / etl / frontend - from node.js / phonegap / that has some utility.

Currently :

* Tooling / Phonegap application for DogCatcher's exported OPML Feed.

* node.js / pouchdb backend

Dependencies :

* node.js / iojs

* npm

* cordova / phonegap

* pouchdb
